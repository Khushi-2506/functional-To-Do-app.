# functional-To-Do-app.
To-Do List App with animations, themes, drag &amp; drop, and localStorage.
This project is a cute and aesthetic To-Do List made with HTML, CSS, and JavaScript.  
It includes:

✔ User Profile  
✔ Statistics Panel  
✔ Themes  
✔ Search & Filters  
✔ Drag & Drop  
✔ Emojis and Stickers  
✔ Smooth Animated Background  
✔ LocalStorage (data saved even after refresh)

## Files Included
- index.html
- style.css
- script.js

## How to Run
Just open index.html in any browser.
